# 模型删除

> ```url
> POST /api/convert/bim-file/delete
> ```

## 说明

把模型文件从当前列表中删除，不删除物理文件。

## 参数
Header
| 名称         | 说明     | 类型   |
| ------------ | --------------- |------  |
| Authorization     | Bearer {Token}    | string |

Body
| 名称         | 说明     | 类型   |
| ------------ | --------------- | ------ |
| ids     | 删除模型id数组    | array |


## 响应

| 名称         | 说明     | 类型   |
| ------------ | -------- | ------ |
| code         | 返回码   | string |
| msg          | 提示消息 | string |
| data         | 返回数据 | json   |
| traceId      | 链路id   | string |
| responseTime | 响应时间 | string |

## 返回数据

无

## 请求示例

##### 请求 path

> http://119.39.38.228:9986/api/convert/bim-file/delete

请求参数

```json
["id1","id2"]
```

## 响应示例

```json
{
	"code": "00000",
	"msg": "请求成功",
	"data": {},
	"traceId": "",
	"responseTime": ""
}
```
