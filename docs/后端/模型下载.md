# 文件下载

> ```url
> GET /api/oss/download
> ```

## 说明

根据文件id下载文件。

## 参数
Header
| 名称         | 说明     | 类型   |
| ------------ | --------------- |------  |
| Authorization     | Bearer {Token}    | string |

Body
| 名称         | 说明     | 类型   |
| ------------ | --------------- | ------ |
| fileUploadRecordId     | 模型文件id   | string |



## 响应

| 名称         | 说明     | 类型   |
| ------------ | -------- | ------ |
| code         | 返回码   | string |
| msg          | 提示消息 | string |
| data         | 返回数据 | json   |
| traceId      | 链路id   | string |
| responseTime | 响应时间 | string |

## 返回数据

文件流

## 请求示例

##### 请求 path

> http://119.39.38.228:9986/api/oss/download?fileUploadRecordId={文件id}
